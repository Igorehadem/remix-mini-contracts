# Remix Mini Contracts

Tiny Solidity examples for learning and deploying on Base or Base Sepolia.

## Includes
- Ping.sol
- Greeter.sol
- Counter.sol
- TinyStorage

Each contract can be compiled in Remix (Solidity 0.8.25) and deployed directly.
